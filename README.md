# Simple E-Voting System (C++)

Description
A console-based e-voting system developed in C++ using file handling.  
The system allows users to register, log in securely, and cast their vote while ensuring that each voter can vote only once.  
All voter data and election results are stored persistently using files.

Features
- Voter Registration System  
- Secure Login Authentication  
- One Person One Vote Enforcement  
- File Handling for Data Storage  
- Persistent Vote Storage (results are saved even after program exit)  
- Simple and User-Friendly Console Interface
